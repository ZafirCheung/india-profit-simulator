## 🚀 三个新增核心功能

### 1. 获客渠道拆分 ⭐⭐⭐
**功能**：区分付费渠道（Google/FB/KOL）和推荐渠道（老带新），分别计算CAC

**新增字段**：
- 付费渠道占比：0-100%（默认40%）
- 付费渠道CAC：₹50-800（默认₹180）
- 推荐渠道CAC：₹20-200（默认₹60）
- 加权平均CAC：自动计算（只读显示）

**计算逻辑**：
```
付费预算 = 总预算 × 付费渠道占比
推荐预算 = 总预算 × (1 - 付费渠道占比)

付费用户数 = 付费预算 / 付费CAC
推荐用户数 = 推荐预算 / 推荐CAC

加权平均CAC = 总预算 / (付费用户数 + 推荐用户数)
```

**效果**：
- 基准场景：40%付费(₹180) + 60%推荐(₹60) = 加权CAC约₹108
- 相比单一CAC ₹90，更真实反映混合渠道成本
- 可以单独优化两个渠道的投放策略

---

### 2. 促销成本细化 ⭐⭐⭐
**功能**：将"促销成本"拆分为4个子项，精准控制每一类成本

**新增字段**：
- 首充奖金：5-20%（默认10%）
- 每日返水：1-8%（默认3%）
- 活动赠金：1-6%（默认2%）
- VIP特权：0.5-5%（默认1.5%）
- 总促销成本：自动求和（只读显示）

**计算逻辑**：
```
总促销成本 = 首充奖金 + 每日返水 + 活动赠金 + VIP特权
促销支出 = 实际收入 × 总促销成本
```

**效果**：
- 核心拆解表格新增"促销成本细分"区域
- 可以看到每一项的具体金额和占收入比例
- 智能建议：首充>15% → 羊毛党过多

### 3. 模式切换按钮 ⭐⭐
**功能**：一键切换社交娱乐/真金/自定义三种业务模式

**位置**：顶部工具栏，在"基准/保守/增长"按钮左侧

**三种模式**：

| 参数 | 社交娱乐模式 | 真金模式 | 自定义模式 |
|------|------------|---------|-----------|
| 提现率 | 0% | 68% | 手动调 |
| 广告ARPDAU | ₹1.5 | ₹0.15 | 手动调 |
| 付费率 | 3.5% | 5.2% | 手动调 |
| ARPPU | ₹500 | ₹1000 | 手动调 |
| 欺诈损失 | 0.5% | 1.2% | 手动调 |
| 税费 | 3% | 8% | 手动调 |

**效果**：
- 社交模式：自动隐藏提现相关，突出广告收入
- 真金模式：自动显示提现率、现金流指标
- 切换时显示Toast提示："已切换到XXX模式"
