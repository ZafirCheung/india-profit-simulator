# 印度盈利模拟器更新说明

## 📊 新增核心指标（标记 ⭐）

### 一、获客与转化板块

**1. 首充转化率** (First Deposit Rate)
- 范围：5-60%
- 默认值：基准 28% / 保守 18% / 增长 35%
- 说明：激活用户中完成首次充值的比例，最关键的漏斗指标
- 为什么重要：这是从免费用户到付费用户的关键跳跃，决定了获客ROI

**2. 首充金额** (First Deposit Amount)
- 范围：₹100-5,000
- 默认值：基准 ₹800 / 保守 ₹500 / 增长 ₹1,200
- 说明：用户首次充值的平均金额
- 为什么重要：首充是用户心理门槛最高的时刻，金额设置直接影响转化率

---

### 二、变现板块

**3. 月充值次数** (Recharge Frequency)
- 范围：1-10次
- 默认值：基准 2.8次 / 保守 2.0次 / 增长 3.5次
- 说明：付费用户每月平均充值次数
- 为什么重要：多次小额充值比单次大额充值更健康，说明用户粘性强

**4. 大R占比** (Whale Rate)
- 范围：0.5-20%
- 默认值：基准 5.5% / 保守 3.0% / 增长 7.2%
- 说明：月充值>₹10,000的高价值用户占付费用户的比例
- 为什么重要：印度市场呈现极端的二八分布，3-8%的大R贡献50-70%收入

**5. 大R月ARPPU** (Whale ARPPU)
- 范围：₹5,000-50,000
- 默认值：基准 ₹8,500 / 保守 ₹5,500 / 增长 ₹12,000
- 说明：高价值用户的月均充值金额
- 为什么重要：需要单独建模，因为他们的LTV和流失成本远高于普通用户

**6. 提现率 ⭐⭐⭐** (Withdraw Rate)
- 范围：40-95%
- 默认值：基准 68% / 保守 72% / 增长 65%
- 说明：提现金额/充值金额，**真金游戏最核心的盈利指标**
- 为什么重要：
  - 60-75% = 健康（平台留存25-40%作为利润）
  - >80% = 玩家赢太多，House Edge过低
  - <50% = 提现门槛太高，影响口碑
- 直接显示为仪表盘指标卡片

---

### 三、成本板块

**7. 支付成功率 ⭐⭐⭐** (Payment Success Rate)
- 范围：60-95%
- 默认值：基准 78% / 保守 72% / 增长 82%
- 说明：印度UPI支付成功率，行业标准70-85%
- 为什么重要：
  - 每降低5%成功率 = 损失3-4%付费转化
  - 每月损失金额 = 总收入 × (1 - 成功率)
  - 解决方案：接入3-5家支付网关，失败自动切换
- 直接显示为仪表盘指标卡片

**8. 每笔提现审核成本** (Withdraw Process Cost)
- 范围：₹0-50
- 默认值：基准 ₹8 / 保守 ₹10 / 增长 ₹7
- 说明：KYC审核+反欺诈人工成本
- 为什么重要：印度市场需要Aadhaar/PAN卡验证，人工审核成本不可忽视

**9. 欺诈损失率 ⭐** (Fraud Loss Rate)
- 范围：0-10%
- 默认值：基准 1.2% / 保守 1.8% / 增长 0.9%
- 说明：刷单、羊毛党、欺诈造成的损失占收入比例
- 为什么重要：印度市场羊毛党猖獗，行业标准0.5-2%

## 🔧 计算逻辑优化

### 1. 分层付费用户计算
```
总付费用户 = MAU × 付费率
大R用户 = 总付费用户 × 大R占比
普通用户 = 总付费用户 - 大R用户

IAP收入 = 普通用户 × ARPPU + 大R用户 × 大R ARPPU
```

### 2. 首充收入单独计算
```
首充用户 = 新增活跃 × 首充转化率
首充收入 = 首充用户 × 首充金额
总收入 = IAP收入 + 首充收入 + 广告 + 订阅 + B2B
```

### 3. 支付成功率影响
```
总收入（Gross Revenue）= 所有收入之和
实际收入（Actual Revenue）= 总收入 × 支付成功率
支付损失（Payment Loss）= 总收入 - 实际收入
```

### 4. 提现与现金流
```
总充值额 = IAP收入 + 首充收入
提现金额 = 总充值额 × 提现率
净现金流 = 总充值额 - 提现金额
```

### 5. 新增成本项
```
提现审核成本 = 付费用户数 × 0.5 × 每笔审核成本
  （假设50%的付费用户会提现）

欺诈损失 = 实际收入 × 欺诈损失率

总成本 = 收入相关成本 + 云成本 + 合规成本 + 提现审核成本 + 
         欺诈损失 + 营销预算 + 固定运营成本
```
