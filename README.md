## ✨ 核心功能

- **真金模式专注**：删除社交娱乐相关字段（广告、订阅、B2B）
- **GGR/NGR指标**：行业标准的Gross Gaming Revenue和净博彩收入
- **House Edge显示**：实时显示平台留存比例
- **智能策略提示**：5-7条分级预警系统
- **双币种显示**：所有金额自动显示卢比+美元（1美元=₹83）
- **核心拆解**：25行数据分4个板块清晰展示

## 📊 主要指标

### 收入指标
- IAP虚拟商品收入（充值）
- 首充收入
- GGR（净现金流入）
- GGR率

### 用户指标
- MAU / DAU
- 付费用户（普通/鲸鱼）
- 首充转化率
- 鲸鱼用户贡献占比

### 盈利指标
- 月净利润
- 利润率
- LTV/CAC
- 盈亏平衡CAC
- 实际House Edge

### 现金流指标
- 总充值额
- 提现金额
- 净现金流入（GGR）
- GGR率
